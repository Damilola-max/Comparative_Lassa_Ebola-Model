# Comparative Lassa Ebola Model

This repository contains all the components required for the Comparative Lassa and Ebola Model project.

## Directory Structure

- `data/`
  - `raw/`
  - `processed/`
  - `embeddings/`
- `results/`
  - `figures/`
- `models/`
  - `esm_weights/`

## Description of Folder Structure

- `data/raw/`: Contains raw datasets.
- `data/processed/`: Contains processed datasets.
- `data/embeddings/`: Contains embeddings used in the models.
- `results/figures/`: Contains figures generated from results.
- `models/esm_weights/`: Contains weights for models.

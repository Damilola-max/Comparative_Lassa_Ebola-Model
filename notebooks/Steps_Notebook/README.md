# Steps Notebook Folder

This folder contains sequential notebooks designed for comparative analysis of the Lassa and Ebola virus models. The notebooks guide users through different stages of the analysis, including data exploration, model fitting, and result visualization.

## Purpose
The primary objective of these notebooks is to provide a structured and easy-to-follow approach to understanding the dynamics of Lassa and Ebola virus models using computational methods.

## How to Use the Notebooks
1. **Environment Setup**: Ensure you have the required libraries and dependencies installed.
2. **Sequential Execution**: It's crucial to run the notebooks in the order they are presented to maintain the flow of data and results.
3. **Understanding Outputs**: Each notebook includes detailed explanations on the output graphs and key takeaways to facilitate interpretation.

Feel free to explore and adapt the notebooks for your own research needs!
